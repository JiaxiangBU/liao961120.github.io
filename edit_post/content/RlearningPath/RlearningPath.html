Course: R語言與資料科學導論

課程起手式

106學年上學期，我最喜歡、也意外收穫最多的課是謝舒凱老師開設的R語言與資料科學導論，或許會是我大學生涯最喜的課吧。

起初是抱著學好R語言(加上通識快修不完)的心情去加簽這門課的，但事實上，這門課R語言的語法教得非常少。然正是這種不聚焦在程式語法上的課程安排，讓我把R學得非常好。如果課程從頭到尾都在教R語言，到期末我一定會受不了越顯複雜的語法1，然後就像很多科目一樣隨意應付。我在這門課收穫最大的，不是課堂學到的技巧，而是老師、助教們傳達的概念、想法：

  • R的生態圈2以及 Open Source 的概念

  • 資料科學與 Story telling: Coding 於資料科學中的角色不同於其在工程上的角色，亦及 Aesthetic Coding 與 Functional Coding 的概念。資料科學不僅包含傳統的 Functional Coding，也包含大量的 Aesthetic Coding，用於傳達資料科學上的發現

  • Reproducibility: Rmarkdown{:target=“_blank“}(.Rmd) 是我見過最強大的文件格式，可以輸出成網頁、投影片、PDF、Word、Markdown等。這讓使用者可以在一份文件中做事情(Rmd)，並依據所需輸出成各種文件，而不用剪剪貼貼(從 Word 貼到 PPT)，也省下排版花費的心力。例如，要報告可輸出成投影片、要列印可輸出成PDF、要放到網路上可輸出成網頁。這讓使用者能有效率的管理與記錄工作流程，是 Reproducibility{:target=”_blank“} 的基礎。

  • 文本分析: 我本來對這部分最沒有興趣，直到我意識到文本資料是研究人類行為、社會與文化最重要的資料來源之一3,4
